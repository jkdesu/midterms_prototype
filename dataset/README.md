# Dataset — Volatility of Everyday Life

Reference data for banana price comparison (Tokyo vs NYC).

The web app fetches the live USD/JPY rate from [Frankfurter API](https://api.frankfurter.app/) in the browser. Banana prices are embedded in `main.js`.

| File | Description |
|------|-------------|
| `exchange-rates.json` | Cached USD→JPY (optional reference) |
| `banana-prices.json` | Reference prices (optional) |
